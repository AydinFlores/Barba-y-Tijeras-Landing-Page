# Barba-y-Tijeras-Landing-Page
Landing page para una peluqueria urbana con enfoque en servicios, galeria, equipo y reservas. Proyecto estatico en HTML/CSS/JS.

## Vista rapida
- Hero con propuesta de valor y CTA.
- Secciones de servicios, galeria y equipo.
- Formulario de reservas (front-only).
- Footer con datos de contacto.

## Estructura
- `index.html`
- `css/styles.css`
- `js/main.js`
- `assets/`

## Uso
1. Abrir `index.html` en el navegador.
2. Editar textos e imagenes en `index.html` y `assets/`.

## Personalizacion
- Cambiar colores y tipografia en `css/styles.css`.
- Ajustar interacciones en `js/main.js`.
